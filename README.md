# **Assignment: Student Information Management App using Flutter and MySQL**
## Course
Mobile Application Development

## Assignment Title
Develop a Student Information Management Application with Backend Database Integration 

## Objective
The objective of this assignment is to develop a mobile application using Flutter that allows users to:
1. Enter student information through a form.
2. Store the submitted information in a backend MySQL database using PHP APIs.
3. Retrieve and display the stored information from the database.

## Problem Statement
Design and develop a Flutter mobile application that collects student details through a form
and stores them in a MySQL database hosted on XAMPP. The application should also
provide functionality to fetch and display all stored records from the database.
